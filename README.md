# سيرفر إدارة وأسعار الهواتف والمجهزين في العراق

مشروع لإنشاء خلفية برمجية (Backend API) متكاملة لإدارة أسعار الهواتف الذكية وتحديثها مركزيًا، مع ربطها بقاعدة بيانات سحابية متطورة.

## التقنيات المستخدمة:
* Python & FastAPI: لبناء واجهة برمجة تطبيقات (API) سريعة البرق.
* Supabase & PostgreSQL: قاعدة البيانات المركزية لإدارة وتخزين البيانات العلائقية بشكل مستقر.
* SQL Indexes: استخدام الفهارس لضمان جلب البيانات وتصفيتها بلمح البصر حتى مع ملايين الأجهزة.

## هيكل قاعدة البيانات:
1. جدول المجهزين (Suppliers): لإدارة تجار الجملة وعناوينهم وأرقام هواتفهم.
2. جدول المنتجات (Products): لإدارة الأجهزة، الماركات، ومتابعة الأسعار الحالية وتحديثها بشكل حي.
